# 05-菜单与路由配置

## 1. 路由配置 (router/index.ts)
在元数据模块（`/metadata`）下新增日志明细路由：

```typescript
{
  path: '/metadata/log',
  name: 'LogDetail',
  component: () => import('@/views/metadata/LogDetail.vue'),
  meta: { title: '日志明细' }
}
```

## 2. 菜单配置 (SidebarLayout.vue)

### 2.1 顶部水平导航栏
在元数据模块的 `el-sub-menu` 中添加菜单项：

```html
<el-sub-menu index="/metadata">
  <template #title>元数据</template>
  <el-menu-item index="/metadata/field">事件属性</el-menu-item>
  <el-menu-item index="/metadata/event">元事件</el-menu-item>
  <el-menu-item index="/metadata/dimension">维度</el-menu-item>
  <el-menu-item index="/metadata/log">日志明细</el-menu-item> <!-- 新增 -->
</el-sub-menu>
```

### 2.2 侧边垂直导航栏
在 `subMenuItems` 计算属性中，为元数据模块添加日志明细项：

```typescript
if (module === '/metadata') {
  return [
    { path: '/metadata/event', title: '元事件', icon: 'Collection' },
    { path: '/metadata/field', title: '事件属性', icon: 'List' },
    { path: '/metadata/dimension', title: '维度', icon: 'Management' },
    { path: '/metadata/log', title: '日志明细', icon: 'Memo' } // 新增
  ]
}
```

## 3. 图标说明
- 使用 Element Plus 的 `Memo` 图标作为日志明细的视觉标识。
- 图标已在 `main.ts` 中全局注册，可直接通过组件名引用。
