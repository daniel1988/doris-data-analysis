# 05-看板管理开发文档 (修订版)

看板管理模块负责将分析结果固化并以仪表盘形式展示。该页面共享系统的统一侧边栏布局。

## 1. 模块概览

- **页面**: 看板详情 (`DashboardDetail.vue`)。
- **结构**: 
  - **主体区**: 展示仪表盘统计卡片及各种图表（趋势、分布、漏斗、地域等）。
  - **右侧侧边栏**: 用于在看板内进行特定的管理操作（如切换分组、添加报表等），或通过顶部导航进行全局切换。

## 2. 核心功能设计

### 2.1 看板展示 (Dashboard Display)
- **展示方式**: 响应式宫格式布局。
- **顶部功能栏**: 
  - 刷新、分享、导出 PDF/图片、编辑模式切换。
- **交互**: 全屏查看、手动刷新图表数据。

### 2.2 看板分组与列表
- **功能**: 看板支持按业务线或个人/共享进行分组。
- **展示**: 分组列表通常展示在左侧菜单栏的“看板管理”二级菜单中，或在页面内的侧边导航中。

### 2.3 看板设计器 (Dashboard Designer)
- **配置项**:
  - 添加图表：从已保存的分析（Saved Analysis）中选择。
  - 调整位置：拖拽调整卡片大小和顺序。

## 3. 开发步骤建议

### 第一阶段：看板基础结构
1. **统一布局接入**: 确保看板页面挂载在 `SidebarLayout` 下。
2. **数据展示卡片**: 实现 UV/PV 等统计卡片组件。

### 第二阶段：图表集成
1. **ECharts 集成**: 实现多种图表类型的封装展示。
2. **响应式适配**: 确保在不同屏幕尺寸下，图表卡片能自动排列。

### 第三阶段：管理逻辑
1. **分组切换**: 在页面侧边或通过菜单实现看板分组的快速切换。
2. **操作逻辑**: 实现刷新、导出等基础功能。
