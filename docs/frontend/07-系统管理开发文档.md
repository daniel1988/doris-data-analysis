# 07-系统管理开发文档

系统管理模块负责用户、角色、权限、以及多租户/项目空间的管理。

## 1. 模块概览

- **主要页面**: 用户管理 (`UserList.vue`), 角色与权限 (`RolePermission.vue`), 租户管理 (`TenantList.vue`), 操作日志 (`AuditLog.vue`)。
- **技术要点**:
  - RBAC (Role-Based Access Control) 模型。
  - 动态路由与按钮级权限控制。
  - 多租户数据隔离配置。

## 2. 核心功能设计

### 2.1 用户与角色 (User & Role)
- **管理功能**: 用户账号增删改查、重置密码、分配角色。
- **权限配置**: 为角色勾选菜单树、功能按钮。

### 2.2 租户/项目管理 (Tenant/Project)
- **概念**: 一个租户（如：某公司）下可有多个项目（如：APP A, 小程序 B）。
- **配置**: 项目 AppKey 生成、Doris 对应数据库名映射。

### 2.3 操作日志 (Audit Log)
- **记录**: 谁在什么时候做了什么操作（如：修改了某看板、导出了某数据）。

## 3. 开发步骤建议

### 第一阶段：基础用户管理
1. **用户列表**: 实现基础的账号管理功能。
2. **个人中心**: 修改头像、个人信息、密码。

### 第二阶段：权限系统 (RBAC)
1. **角色管理**: 实现角色定义与菜单权限关联。
2. **权限指令/组件**: 开发 Vue 指令（如 `v-permission`）实现按钮级隐藏。

### 第三阶段：多租户与全局配置
1. **租户管理**: 实现租户创建与项目空间初始化。
2. **系统参数**: 配置系统名称、Logo、全局分析参数等。

## 4. 数据契约示例

### 4.1 用户信息 (GET /api/v1/system/user/info)
```json
{
  "username": "admin",
  "roles": ["super_admin"],
  "permissions": ["analytics:view", "system:manage"],
  "tenant_id": "t_001"
}
```
