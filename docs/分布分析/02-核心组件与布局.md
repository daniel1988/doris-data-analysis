# 02-核心组件与布局

分布分析页面将沿用项目统一的分析类页面布局架构，采用“左配置-右展示”的模式。

## 1. 页面布局 (ScatterAnalysis.vue)

采用 `ScatterAnalysisLayout`（待创建）作为基础容器，包含以下区域：

### 1.1 左侧配置区 (Config Pane)
- **指标配置 (ScatterMetricsList)**：仅支持配置**单个**指标。包含事件选择、度量属性选择、指标过滤。
- **分布设置 (ScatterSettingsCard)**：配置分布类型（离散、自动区间、自定义区间等）。
- **全局过滤 (GlobalFiltersCard)**：通用的全局筛选组件。
- **分组配置 (GlobalGroupsCard)**：通用的分组维度组件。

### 1.2 底部操作栏 (Action Footer)
- **开始分析**按钮。
- **保存报表**按钮。
- **重置**按钮。

### 1.3 右侧展示区 (Result Pane)
- **时间与粒度控制 (TimeControls)**：选择分析的时间范围。
- **结果区域 (ResultView)**：
  - 顶部：可视化切换（柱状图、表格）、另存为图片、查看 SQL。
  - 中部：图表展示区（柱状图为主）。
  - 底部：数据明细表格。

## 2. 核心组件说明

### 2.1 ScatterMetricsList.vue
- 专门为分布分析设计的指标配置组件。
- 内部引用 `MetricItem.vue`，但限制只能添加一个指标。
- 需要通过 props 隐藏“自定义公式”和“复制”功能。

### 2.2 ScatterSettingsCard.vue
- 新增组件，用于配置分布的具体参数。
- 包含：
  - 分布类型切换（单选框）。
  - 自定义区间配置（动态增减区间行，包含 min/max 输入）。
  - 分桶数量设置（针对自动区间）。

### 2.3 ScatterResultView.vue
- 渲染分布结果。
- 柱状图需要特殊处理：横轴通常是区间描述字符串，纵轴是人数。
- 支持点击柱子下钻查看用户列表（可选）。
