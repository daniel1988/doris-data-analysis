# 留存分析 - 页面布局优化方案 (热力图矩阵版)

根据业务需求，留存分析模块将取消图表展示，专注于**留存矩阵表格**的深度优化。本方案旨在通过增强表格的视觉表现力和交互性，提升大数据量下的分析体验。

---

## 1. 核心设计理念

-   **视觉直观性**：利用色阶热力图替代趋势图，快速识别留存率波峰与波谷。
-   **信息高密度**：在有限的屏幕空间内展示更多的维度、日期和留存数据点。
-   **操作即时性**：通过固定行列和快捷切换，确保在大表格中不丢失上下文。

---

## 2. 布局优化细节

### 2.1 结果区域重构
-   **移除图表区**：移除原有的 `chart-section`，将全部垂直空间分配给表格。
-   **全屏/自适应高度**：表格容器应填充 `main-container` 的剩余高度，避免出现双滚动条。

### 2.2 增强型留存矩阵表格
-   **固定行列 (Sticky Layout)**：
    -   **固定表头**：留存天数（Day 1, Day 7...）在垂直滚动时始终可见。
    -   **固定前缀列**：分组维度、初始日期、初始人数（Day 0）在水平滚动时始终可见。
-   **热力图单元格 (Heatmap Cells)**：
    -   根据留存率（0% - 100%）动态计算背景色深浅（如：`rgba(64, 158, 255, alpha)`）。
    -   留存率越高，背景色越深；低留存率使用浅色或灰色，减少视觉干扰。
-   **双重数值展示**：
    -   提供切换按钮：**展示百分比 (%)** / **展示绝对人数 (N)** / **两者同时展示**。

### 2.3 工具栏与交互
-   **色阶对照图例**：在表格右上方显示色阶条，标识不同颜色的留存率区间。
-   **数据过滤与搜索**：支持对表格内的“初始日期”或“分组名称”进行快速筛选。
-   **导出增强**：支持将带颜色标记的表格导出为 Excel 或图片（用于周报/月报）。

---

## 3. 视觉规范

| 元素 | 优化建议 |
| :--- | :--- |
| **Day 0 (基准列)** | 采用加粗字体和深灰色背景，强调其作为分母的地位。 |
| **数值字体** | 使用等宽字体 (Monospace)，确保数值对齐，提升扫视效率。 |
| **单元格 Padding** | 适当缩小 Padding（如 `8px 12px`），使矩阵看起来更紧凑。 |
| **无数据处理** | 未发生日期或空数据点使用 `-` 或斜线填充，并使用低对比度颜色。 |
