# 02-数据模型设计

数据看板的核心由“看板配置”和“看板项（组件）”两部分组成。

## 1. 看板 (Dashboard)
存储看板的基本配置信息。

- `id`: 唯一标识 (uint, primary key)
- `project_alias`: 项目别名 (string)
- `name`: 看板内部名称 (string)
- `display_name`: 看板显示名称 (string)
- `description`: 看板描述 (string)
- `category`: 看板分类（如：overview, analytical, custom）
- `layout_type`: 布局模式（grid, flex, custom）
- `grid_config`: 全局网格配置 (JSON)
- `theme`: 主题设置（light, dark, auto）
- `refresh_interval`: 自动刷新频率（秒，0为不刷新）
- `filters`: 全局筛选器配置 (JSON)
- `variables`: 仪表盘变量 (JSON)
- `status`: 状态（draft, published, archived）
- `owner_id`: 创建者 ID (int)
- `create_time`: 创建时间 (time.Time)
- `update_time`: 更新时间 (time.Time)

## 2. 看板项 (DashboardItem)
存储看板内各组件的关联报表及布局信息。

- `id`: 唯一标识 (uint, primary key)
- `dashboard_id`: 所属看板 ID (uint, foreign key)
- `report_id`: 关联的报表 ID (uint)
- `type`: 组件类型（chart, table, mixed, stat, map）
- `title`: 组件覆盖标题 (string)
- `position_x`: 网格 X 坐标
- `position_y`: 网格 Y 坐标
- `width`: 网格宽度 (units)
- `height`: 网格高度 (units)
- `z_index`: 层级深度
- `config_override`: 报表配置覆盖 (JSON)
- `is_visible`: 是否可见
- `create_time`: 创建时间
- `update_time`: 更新时间
