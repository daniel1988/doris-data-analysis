# 01-功能概述与核心逻辑

## 1. 目标
实现一个灵活的数据看板模型，允许用户将各种分析模型（事件、留存、漏斗、属性、分布分析）中保存的报表直接组合成一个数据看板。

## 2. 核心功能
- **报表组合**：在保存报表（事件、留存、漏斗、属性、分布等）时，支持直接将其添加至指定看板。
- **动态菜单展示**：新建的看板将自动作为侧边栏菜单项展示在“看板”页面，支持基础的分组管理。
- **全局筛选器 (Global Filters)**：看板支持配置全局筛选条件（如时间范围、渠道、版本等），统一控制看板内所有报表的数据过滤。
- **布局管理**：采用响应式栅格系统，支持看板内报表组件的拖拽排版、缩放及层级调整。
- **自动刷新**：支持配置看板自动刷新频率（如每5分钟刷新一次），适用于监控场景。
- **多主题支持**：支持浅色/深色模式切换，适配不同展示环境。
